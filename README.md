## <a name="french-description"></a> Description en français

Ces exercices proviennent du cours **C12 : Développer en Front-end, S4 : Le langage JSX**. Ils illustrent l’utilisation de JSX dans React. 

1. **Hello React avec JSX**  
   Affiche un paragraphe `<p>` contenant le texte "Hello React" en utilisant JSX et ReactDOM.

2. **Créer une liste de cinq éléments**  
   Utilisation de `<ul>` et `<li>` pour créer une liste simple.

3. **Ajout d’attributs**  
   Ajouter des attributs `id`, `className` et `style` aux éléments JSX pour appliquer des styles CSS.

4. **Instructions JavaScript dans JSX**  
   Utiliser des variables et expressions JavaScript entre accolades `{}` pour définir des styles ou afficher des valeurs dynamiques.

5. **Liste dynamique à partir d’un tableau**  
   Parcourir un tableau avec `.map()` pour créer automatiquement les éléments `<li>`.

6. **Fonctions JSX**  
   Créer des fonctions qui retournent du JSX, permettant de réutiliser le code et de transmettre des attributs via `props`.

7. **Composants fonctionnels et props**  
   Transmettre des données comme `elems` ou `color` à un composant fonctionnel et les utiliser dans le JSX.

8. **Création de composants avec classes**  
   Définir des composants React en utilisant des classes qui héritent de `React.Component`. Permet d’utiliser `this.props` et d’ajouter des méthodes ou un `state` si nécessaire.

9. **Fragment et parent unique**  
   Utiliser `<React.Fragment>` pour retourner plusieurs éléments JSX sans ajouter un parent `<div>` supplémentaire.

10. **Parenthèses et commentaires dans JSX**  
    Utiliser des parenthèses pour JSX multi-lignes et `{/* commentaires */}` pour commenter des éléments JSX.

11. **Expressions conditionnelles**  
    Afficher ou masquer un élément selon une condition avec `{ condition ? <Element /> : null }`.

Ces exercices montrent les bases de JSX et comment créer des composants modulaires et dynamiques en React.

## <a name="english-description"></a> Description in English

These exercises come from the course **C12: Front-end Development, S4: JSX Language**. They illustrate how to use JSX in React.

1. **Hello React with JSX**  
   Displays a `<p>` paragraph containing "Hello React" using JSX and ReactDOM.

2. **Create a list of five items**  
   Use `<ul>` and `<li>` to create a simple list.

3. **Add attributes**  
   Add `id`, `className`, and `style` to JSX elements to apply CSS styles.

4. **JavaScript expressions in JSX**  
   Use variables and JavaScript expressions inside `{}` to define styles or display dynamic values.

5. **Dynamic list from an array**  
   Iterate over an array with `.map()` to automatically create `<li>` elements.

6. **JSX functions**  
   Create functions that return JSX to reuse code and pass attributes via `props`.

7. **Functional components and props**  
   Pass data like `elems` or `color` to a functional component and use them in JSX.

8. **Class-based components**  
   Define React components using classes that extend `React.Component`. Allows using `this.props` and adding methods or `state` if needed.

9. **Fragment and single parent**  
   Use `<React.Fragment>` to return multiple JSX elements without adding an extra `<div>` parent.

10. **Parentheses and comments in JSX**  
    Use parentheses for multi-line JSX and `{/* comments */}` to comment JSX elements.

11. **Conditional expressions**  
    Show or hide an element based on a condition with `{ condition ? <Element /> : null }`.

These exercises cover JSX basics and how to create modular, dynamic React components.
